# agent-skills

> 个人 Agent Skill 仓库，持续孵化中 🚀

## 目录结构

```
skills/
└── <skill-name>/
    ├── SKILL.md          # Skill 主文件（元数据 + 使用说明）
    ├── scripts/          # 辅助脚本（可选）
    └── references/       # 参考文档（可选）
```

每个 Skill 目录下的 `SKILL.md` 包含完整的使用说明和参考文档索引。
